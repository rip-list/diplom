# Flutter Diploma Project

## Overview
This Flutter diploma project serves as a comprehensive showcase of the capabilities of the Flutter framework. It includes the development of a cross-platform application for both iOS and Android, demonstrating the fundamental principles of Flutter, state management, and responsive design.

## Features
- **Cross-Platform Support**: Runs seamlessly on both iOS and Android.
- **State Management**: Utilizes modern state management techniques such as Provider and Riverpod.
- **Responsive UI**: Adapts to various screen sizes with Flexbox layouts and MediaQuery.


## Installation
To get started, clone the repository and run the following commands:

```bash
flutter pub get
flutter run
```

## Usage
After launching the application, you will be greeted with a user-friendly interface. Follow the on-screen instructions to navigate through the features.
